🎯 WHAT EACH TOOL DOES:

📊 WEBSITE ANALYZER
   → Paste any URL
   → Get SEO score, performance score, conversion score
   → Identifies 5 key issues
   → Provides 5 improvement recommendations
   → Competitor analysis included

   API: POST /api/ai/analyze-website
   Input: { url: "https://example.com" }

✨ CONTENT GENERATOR
   → Enter business name + industry
   → Choose content type:
     • Blog Post Ideas (5 ideas with keywords)
     • Social Media Posts (10 posts)
     • Meta Tags (title + descriptions for pages)
     • Email Sequences (3-email nurture series)
     • FAQ Section (10 Q&A pairs)

   API: POST /api/ai/generate-content
   Input: { businessName, niche, contentType }

💰 SMART QUOTE GENERATOR
   → Analyzes your project needs
   → Recommends: Starter/Professional/Enterprise
   → Returns: Price, timeline, cost breakdown, ROI
   → Suggests upsells

   API: POST /api/ai/smart-quote
   Input: { businessName, niche, services, hasWebsite, currentProblems }

📋 PROJECT SCOPING
   → Generates complete project roadmap
   → Creates project phases & timelines
   → Lists deliverables
   → Identifies risks
   → Defines success metrics
   → Asks clarifying questions

   API: POST /api/ai/onboarding
   Input: { businessName, niche, currentWebsite, goals }

💬 AI CHAT ASSISTANT (Ready to deploy)
   → 24/7 customer support
   → Answers questions about services
   → Qualifies leads
   → Guides to booking

   API: POST /api/ai/chat
   Input: { message, context }

💡 EXAMPLE USAGE (JavaScript):

// Analyze a website
const result = await fetch('http://localhost:3000/api/ai/analyze-website', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' })
});
const data = await result.json();
console.log(data.analysis.seoScore);  // 78
console.log(data.analysis.issues);    // Array of 5 issues
console.log(data.analysis.recommendations); // Array of fixes

// Generate content
const content = await fetch('http://localhost:3000/api/ai/generate-content', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    businessName: "Acme Fitness",
    niche: "fitness",
    contentType: "blog"
  })
});
const ideas = await content.json();
console.log(ideas.content); // Array of 5 blog ideas

🔧 TROUBLESHOOTING:

❌ "Could not reach the server"
   ✓ Make sure: npm start is running
   ✓ Check: http://localhost:3000 in browser
   ✓ Try: npx kill-port 3000 (then restart)

❌ "Invalid API key"
   ✓ Verify .env has CLAUDE_API_KEY=sk-ant-...
   ✓ No spaces in the key
   ✓ Key must start with sk-ant-

❌ Slow responses
   ✓ Try Haiku model (faster)
   ✓ Reduce max_tokens in server.js
   ✓ Check internet connection

❌ Weird results
   ✓ Review the prompt templates in server.js
   ✓ Try a different Claude model
   ✓ Add more context to inputs
